關於我

你好！我目前為自己的手上專案開發了自動化排程系統。我熱衷於使用 HTML、CSS 與 google Script 打造高效能且直覺的網頁體驗，並持續探索現代技術如 React 與 Tailwind CSS。

我具備深厚的背景知識，在做事風格上，我極度重視溝通效率。我喜歡教學，享受將複雜的邏輯轉化為淺顯易懂的知識，並堅持「把東西講清楚」是所有成功開發專案的基石。

  • 高效溝通： 能精確捕捉需求，將複雜技術轉化為白話語言。
  • 高品質交付： 堅持撰寫乾淨、可維護的程式碼並提供完整的文件說明。
  • 靈活應變： 豐富的接案經驗讓我能快速適應不同規模與風格的專案開發。
